# interact

## 概要

任意のコマンドを対話モード化するためのコマンド

例えば、このタブではgitしかいじんないんだから打つのめんどいだけだし省略したい、などの問題を解決する

需要はほとんどないとおもう

## 使い方

### インストール

このレポジトリの`interact`を`/usr/local/bin`にぶちこんで、

`chmod +x /usr/local/bin/interact`

する
